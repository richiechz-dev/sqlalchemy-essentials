## Sandbox for practicing SQLAlchemy ORM
This repository is a simple sandbox project for practicing SQLAlchemy ORM in a terminal-based enviroment. It demonstrates basic CRUD (Create, Read, Update, Delete) operations using SQLAlchemy with a SQLite database in memory.

### Tree Structure
```
CRUD_SQLalchemy_Terminal/ 
│├── models.py
│├── database.py
│└── db_test.py
```
### Files Description
- `models.py`: Contains the SQLAlchemy object model for the database tables.
- `database.py`: Manages the database connection and session creation.
- `db_test.py`: Contains functions to perform the creation of the db, and CRUD operations.
